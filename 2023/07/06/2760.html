盒子模型

行元素和块元素

  • 块元素
    • 可以独占一行
    • 块元素默认垂直方向排列
    • 可以设置宽高
    • 一般使用块元素做页面布局
  • 行元素（内联元素，行级元素）
    • 不会独占一行
    • 行元素默认水平方向排列
    • 不能设置宽高

行元素和块元素相互转换

  • display属性
    • display: block;将元素转化为块元素(块元素默认值)
    • display: inline;将块元素转化为行元素(行元素默认值)

隐藏元素

  • display: none;隐藏元素,不占据原本的空间
  • 应用场景: 通过JS控制元素的显示与隐藏

行内块元素

  • display: inline-block;将元素设置为行内块元素
  • 特点
    • 可以水平排列
    • 可以设置宽高

行内块元素空白处理

  • 行内块元素后面多出来的空白,是由于换行符被浏览器解析为了一个空格导致的
  • 处理的办法
    • 将标签后面的换行去掉
    • 将容器(行内块的包含块)的字体大小设置为0,将容器内部的元素字体设置为正常大小(常用)
    • 将容器(行内块的包含块)的display属性设置为table

列表标签

  • 无序列表(ul)
    • 无序列表的每一个子项都应该放在li标签中
    • 无序列表的每一个子项左侧默认会显示一个小圆点
  • 有序列表(ol)
    • 有序列表的每一个子项都应该放在li标签中
    • 有序列表的每一个子项左侧默认会显示一个当前项的序号
  • list-style属性
    • 一般就是将该属性设置为none,不显示默认的样式

盒子模型概念

在CSS中，所有的元素都是包裹在一个个盒子当中。盒子模型定义了盒子的组成部分，这些组成部分合并在一起最终构成了元素在页面的呈现效果

  • 盒子模型的组成
    • 内容区（content）：通过属性width和height设置
    • 内边距（Padding）：内容区外面的空白区域，通过padding相关属性设置
    • 边框（border）：包裹内边距和内容区，通过border相关属性设置
    • 外边距（Margin）：元素和元素间距，通过margin相关属性设置

外边距属性

  • margin-top 设置上外边距
  • margin-bottom 设置下外边距
  • margin-left 设置左外边距
  • margin-right 设置右外边距
  • 复合属性margin可以同时设置多个方向的外边距
    • 单个值 ：上下左右
    • 两个值：上下 左右
    • 三个值：上 左右 下
    • 四个值：上 下 左 右

盒子水平居中

  • margin: 0 auto; 可以将盒子水平方向居中
  • auto可以自动计算容器中剩余的空间，平均的分配到盒子的两边，让盒子实现居中的效果

负margin

  • margin-top设置为负值，元素向上移动
  • margin-left设置为负值,元素向左移动
  • margin-right设置为负值，自身不动，右侧的元素向左移动
  • margin-bottom设置为负值，自身不动，下面的元素向上移动

外边距折叠

相邻的块元素在垂直方向上的外边距不会累加，而是产生重叠现象

  • 如果都是正值，取最大值
  • 如果是一正一负，取累加之后的值
  • 如果都是负值，取绝对值最大

外边距传递

  • 预期：父元素里面的子元素设置margin-top，希望子元素可以跟父元素产生边距
  • 实际：margin-top最终影响了父元素的位置
  • 原因：子元素设置了margin-top，但是找不到父元素的边界，因此将margin-top传递给了父元素
  • 解决办法
    • 给父元素添加padding
    • 给父元素添加border
    • 给父元素设置属性overflow:hidden;

百分比单位

  • 语法：属性：XX%；
  • 注意
    • px是固定单位，百分比是相对单位
    • 子元素设置width和height的百分比，是相对于父元素width和height进行计算
    • 子元素设置margin和padding的百分比，是相对于父元素的width进行计算

盒子垂直居中

  • calc函数：完成数学计算
  • 语法：calc(数学表达式)
  • 计算方式
    • （大盒子的宽度-小盒子的宽度) / 2得出的结果给margin-left

边框属性

包裹内边距和内容区的区域,边框的设置包含四个方向及三个要素

  • 方向：top、bottom、left、right
  • border-方向-width 设置对应方向边框的宽度
  • border-方向-style 设置对应方向边框的样式
    • solid 实线
    • dashed虚线
    • dotted点线
  • border-方向-color 设置对应方向边框的颜色
  • Ctrl + d 可以选中下一个与当前文本匹配的内容，批量修改，按下Esc取消批量操作
  • border-top: 设置上边框
  • border-bottom: 设置下边框
  • border-left: 设置左边框
  • border-right: 设置右边框
  • border：设置四个方向的边框
  • 取值：宽度 样式 颜色
  • 设置边框会影响盒子的尺寸(改变盒子大小)

内边距属性

  • padding-top 设置上内边距
  • padding-bottom 设置下内边距
  • padding-left 设置左内边距
  • padding-right 设置右内边距
  • padding 设置四个方向的边距
  • 注意
    • 设置padding也会影响盒子的尺寸
    • padding区域也会应用背景设置
    • padding可以增加内容区和边框的间距，此处的内容不只是文本，只要包含在标签以后都是元素的内容
    • 如果希望增加子元素和父元素的距离，可以给子元素设置margin，也可以给父元素设置padding

盒子尺寸

  • 盒子可见大小
    • 宽度：width + padding-left + padding-right + border-left + border-right
    • height + padding-top + padding-bottom + border-top + border-bottom
  • 盒子占地大小
    • 宽度：可见宽度 + margin-left + margin-right
    • 高度: 可见高度 + margin-top + margin-bottom

IE盒子（怪异盒子）

又称为怪异盒子，有别于标准盒子的属性设置方式

  • 标准盒子模型的width,height设置的只是内容区的宽度， 如果修改了border或者padding，那么盒子大小会发生改变
  • IE盒子模型width,height设置的是：内容区+内边距+边框,如果修改了border或者padding，总大小不会变，会改变内容区的大小
  • 通过属性box-sizing 可以在两种盒子模型中切换
    • content-box 标准盒子模型
    • border-box IE盒子模型

IE盒子尺寸

可见大小

  • 宽度：width
  • 高度：height

占地大小

  • 宽度：width + margin-left + margin-right
  • 高度：height + margin-top + margin-bottom

重置样式

默认样式举例

  • li默认有圆点或者数字序号
  • a默认有下划线
  • h元素带有上下margin
  • p元素带有上下margin
  • body有默认的一圈margin
  • ul左侧有padding
