10-表格与表单

发表于2023-07-13|更新于2023-07-18
|阅读量:

简单表格

  • table定义表格元素
  • tr表示行
  • td单元格
  • th表头单元格

细分结构的表格

  • thead 定义表格头部
  • tbody 定义表格主体
  • tfoot 定义表格底部统计信息

单元格边框间距与折叠

  • border-spacing 设置单元格之间的距离
  • border-collapse 设置单元格之间的边框是否折叠
    • separate 默认值，分离
    • collapse 折叠

单元格跨行跨列

  • 跨列: colspan
  • 跨行: rowspan

普通文本和密码输入框

  • 普通文本输入框
  • 密码输入框

单选框和复选框

  • 单选框
  • 复选框
  • 注意
    • 一组单选或者复选框的name必须一致
    • checked属性表示当前选项被选中

下拉框

  • select
  • 注意
    • select元素中，放入option元素。每个option就是一个具体的下拉选项
    • option的value是提交数据的值，option标签包裹的内容是给用户看的
    • 可以通过给option添加selected改变默认选项
    • 如果希望下拉框多选，可以给select元素添加multiple属性

文件上传

  • 注意
    • 可以加入multiple属性支持上传多个文件

文本域

  • cols和rows用于设置文本域的可视列数及行数（不好用）,推荐使用width和height
  • textarea标签包裹的内容会显示在页面文本域中
  • resize 设置元素是否可以改变大小
    • both 水平和垂直方向都可以改变
    • horizontal 水平方向可以改变
    • vertical 垂直方向可以改变
    • none 不允许改变

h5新增了很多额外的元素类型

  • 其中一部分后续可以使用UI框架提供更好用的功能以及更美观的样式
  • 类似于手机，邮箱验证，输入数字，字符限制的功能后续使用正则更加严谨方便
  • 完整类型文档：https://developer.mozilla.org/zh-CN/docs/Web/HTML/Element/input

按钮

  • input和button
  • type=submit: 提交表单数据
  • type=reset: 重置表单数据
  • type=button: 普通按钮

表单标签

  • form: 将表单数据提交到服务器
  • 语法
  • 注意
    • action一般就是放服务器接收数据的地址，我们这里可以使用一个html文档接收
    • method可以是get和post。get提交的数据可以在浏览器地址栏中看见，post不可以
    • 如果希望将某个表单数据提交，必须指定这个元素的name属性，否则不会提交!!
    • 在form元素中，点击submit类型的按钮时，就会将form中所有设置name的表单元素值，通过method
    • 在from元素中，点击reset类型的按钮，就是将form内部所有的表单元素重置为初始值

表单属性

  • required必填元素，如果当前元素没有内容，在submit时就不允许提交表单
  • readonly 只读元素。如果当前元素展示了内容，不能修改，只能去看。设置只读的元素数据也会提交
  • disabled 禁用元素。如果元素展示了数据，不能修改，只能看，并且也不希望随着表单提交，设置为禁用

标签元素

  • label 标签元素，为表单元素提供说明的文本
  • 给表单元素设置id值
  • 加上label，将label的for属性设置为指定表单的id值
